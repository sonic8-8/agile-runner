- 적용 예시를 고칠 때 먼저 여는 문서와 같이 갱신할 파일 정리

- 유지 보수 체크리스트, 마지막 확인 질문, 단계 마감 기준 정리

  - 어긋남이 생기면 어디부터 다시 볼지
  - 마지막에 무엇을 다시 확인할지
  를 다시 따라갈 수 있다.
